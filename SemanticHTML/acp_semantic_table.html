A HTML table of Semantic Elements

Semantic Element Use1 Use2
<header> Defines a header for a document or section. Typically contains a heading, navigation links, or site logo.
<nav> Defines a section of navigational links. Used for primary navigation menus, sitemaps, or other navigation structures.
<main> Defines the dominant content of a document. Should only have one <main> element per page.
<section> Defines a generic thematic grouping of content. Used to group related content within the <main>> element.
<article> Defines independent, self-contained content. Commonly used for blog posts, news articles, or forum comments.
<aside> Defines content aside from the page content. Often used for sidebars, related content, or advertisements.
<footer> Defines a footer for a document or section. Typically contains copyright information, author details, or site-wide links.
<figure> Defines self-contained content, often with a caption. Used for images, diagrams, code snippets, or other media.
